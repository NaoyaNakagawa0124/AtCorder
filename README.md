## 競技プログラミングの鉄則 演習問題集

### 競技プログラミングの鉄則 演習問題集とは？
競技プログラミングの鉄則（E869120 が執筆）に対応した自動採点システムです。全部で150問以上のプログラミング問題が収録されています。本リポジトリでは、以下のURLに掲載されている問題を取り組み、進捗を管理しています。

- [競技プログラミングの鉄則 問題一覧](https://atcoder.jp/contests/tessoku-book)
